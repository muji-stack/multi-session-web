監視機能

ステータスチェック

アカウントの状態（正常、ロック、凍結など）を確認する機能です。 定期的なチェックにより、問題を早期発見できます。

手動チェック

単一アカウント:

  1. アカウントカードの「⋮」メニューをクリック
  2. 「ステータスチェック」を選択
  3. 結果がカードに反映されます

複数アカウント:

  1. 「チェック」ページを開く
  2. チェックするアカウントを選択
  3. 「一括チェック」をクリック

自動チェック

設定により、定期的な自動チェックが可能です。

  1. 「設定」→「監視」を開く
  2. 「自動監視を有効にする」をオンにする
  3. チェック間隔を設定（推奨: 1〜6時間）

ステータスの種類

ステータス説明
正常問題なく使用可能
ロック一時的にロックされている状態。認証が必要
凍結アカウントが凍結されている状態
不明チェックに失敗した状態

シャドウバンチェック

シャドウバンとは

アカウントが制限されているにもかかわらず、ユーザー本人にはそれが分かりにくい状態のことです。

  • 投稿が検索結果に表示されない
  • 返信が他のユーザーに見えにくい
  • おすすめに表示されにくい

シャドウバンの種類

  • 検索バン (Search Ban): 投稿が検索結果に表示されない、ハッシュタグ検索でヒットしない
  • ゴーストバン (Ghost Ban): 返信が他のユーザーに表示されない
  • センシティブフラグ: アカウントや投稿にセンシティブフラグが付与

チェック手順

  1. 「チェック」ページを開く
  2. チェックするアカウントを選択
  3. 「シャドウバンチェック」をクリック
  4. 結果を確認

チェック結果の見方

  • ✅ 問題なし: 制限されていません
  • ⚠️ 警告: 軽度の制限の可能性
  • ❌ 制限あり: 制限されています

シャドウバンへの対処

制限が検出された場合:

  1. しばらく投稿を控える（数日〜1週間）
  2. スパム的な行動を避ける
  3. プロフィールを見直す
  4. 問題のある投稿を削除

予防策:

  • 短時間での大量投稿を避ける
  • 同じ内容の繰り返し投稿を避ける
  • 不自然なフォロー/アンフォローを避ける
  • 規約違反コンテンツを投稿しない

アラート通知

問題が検出された場合、以下の方法で通知を受けられます：

  • アプリ内通知
  • デスクトップ通知（設定で有効化）

通知設定

  1. 「設定」→「通知」を開く
  2. 「デスクトップ通知」を有効化
  3. 通知を受け取るイベントを選択
