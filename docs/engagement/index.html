エンゲージメント

一括いいね・リポスト

複数のアカウントから、特定の投稿に対して「いいね」や「リポスト」を一括で行う機能です。

一括エンゲージメントの手順

  1. サイドバーの「エンゲージメント」を開く
  2. アクションを実行するアカウントを選択
  3. 「対象URL」に投稿のURLを入力
  4. アクションタイプを選択：
    • いいね: 選択した投稿にいいね
    • リポスト: 選択した投稿をリポスト
    • 引用リポスト: コメント付きでリポスト
    • ブックマーク: ブックマークに追加
  5. 「実行」をクリック

実行オプション

  • 遅延設定: アクション間の遅延時間を設定（推奨: 3〜10秒）
  • ランダム遅延: より自然な動作のために遅延にランダム変動を追加

注意: 短時間に大量のエンゲージメントを行うとアカウントが制限される可能性があります。

フォロー管理

一括フォロー

複数のアカウントから特定のユーザーをフォローする機能です。

  1. 「エンゲージメント」ページを開く
  2. フォローを実行するアカウントを選択
  3. アクションタイプで「フォロー」を選択
  4. 対象ユーザーのプロフィールURLを入力
  5. 「実行」をクリック

一括アンフォロー

フォロー解除も同様の手順で実行できます。

  • アクションタイプで「アンフォロー」を選択
  • 対象ユーザーのURLを入力して実行

フォロー管理のベストプラクティス

  • 1日あたりの上限: フォロー50〜100人程度を推奨
  • 新規アカウント: より少なく（10〜20人）
  • フォロー間隔: 最低5秒以上を推奨
  • フォロワー比率: フォロー数とフォロワー数のバランスを意識

進捗表示

実行中は以下の情報がリアルタイムで表示されます：

  • 完了したアカウント数 / 総数
  • 成功・失敗の内訳
  • 現在処理中のアカウント
  • エラーメッセージ（発生した場合）
